# ❌⭕ Tic Tac Toe Game (Python GUI)

A simple Tic Tac Toe game built using Python and Tkinter, where two players can play on the same system.

# 💡 Overview

This project was created by following a YouTube tutorial as part of my Python learning journey.
It helped me understand GUI development and how to implement game logic in Python.

# ⚙️ Features
  * 🎮 Two-player game (X vs O)
  * 🖥️ Graphical User Interface using Tkinter
  * 🟩 Winning combination highlight
  * 📢 Winner announcement popup
  * 🔄 Player turn switching
    
# 🛠️ Technologies Used
* Python
* Tkinter (for GUI)
  
# 📚 What I Learned
* Building GUI applications using Tkinter
* Handling button events
* Implementing game logic
* Managing player turns and game state
